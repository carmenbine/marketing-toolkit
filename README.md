# Marketing Toolkit (Vanilla JS)

A tiny set of practical tools:
- UTM Builder (copy-ready links)
- A/B Test quick calculator (two-proportion z-test)
- SEO snippet preview + length guidance
- Email subject line scorer (simple heuristics)

## Run locally
Just open `index.html`.

## Tests
Open DevTools → Console. `test.js` runs automatically and prints "All tests passed."

## Deploy (GitHub Pages)
Settings → Pages → Deploy from branch → `main` / root.
